📊 HTML5 テーブル要素 学習サンプル

table / tr / th / td / thead / tbody / tfoot / caption / colgroup / col の役割と使い方を学びましょう。

① テーブルの全体構造

<table> ─ テーブル全体
<caption> ─ テーブルのタイトル（省略可）
<colgroup> / <col> ─ 列のグループ化（省略可）
<thead> ─ ヘッダー行グループ（省略可）
<tr> ─ 行
<th> ─ 見出しセル
<tbody> ─ ボディ行グループ（省略可・自動補完）
<tr> ─ 行
<td> ─ データセル
<tfoot> ─ フッター行グループ（省略可・最後に1つ）
<tr> ─ 行
<td> ─ データセル（合計など）
⚠️ 試験重要ポイント②：tfoot要素の配置ルール

・tfoot 要素は tbody要素・tr要素よりも後に配置します
・1つしか配置できません

✅ 正しい順序：caption → colgroup → thead → tbody → tfoot

※ HTML4では tfoot を tbody より前に書く仕様でしたが、 HTML Living Standard では 後に配置するよう変更されています。
⚠️ 試験重要ポイント③：終了タグの省略

テーブル要素の多くは終了タグを省略できます。

✅ 終了タグ省略可：tr / th / td / thead / tbody / tfoot / caption / colgroup
❌ 終了タグ省略不可：table / col（空要素）

② table要素 / tr要素 / th要素 / td要素

ポイント
・table：テーブル全体を定義するブロック要素
・tr（Table Row）：行を定義する要素
・th（Table Header）：見出しセル。デフォルトで太字・中央揃え
・td（Table Data）：データセル 
<!-- 最小構成のテーブル -->
<table>
  <tr>
    <th>名前</th>
    <th>年齢</th>
    <th>職業</th>
  </tr>
  <tr>
    <td>山田太郎</td>
    <td>28</td>
    <td>エンジニア</td>
  </tr>
  <tr>
    <td>鈴木花子</td>
    <td>32</td>
    <td>デザイナー</td>
  </tr>
</table>

th要素の scope属性

<!-- scope属性：見出しが行・列どちらに対応するかを示す -->
<table>
  <tr>
    <th scope="col">名前</th>   <!-- 列見出し -->
    <th scope="col">点数</th>
  </tr>
  <tr>
    <th scope="row">山田太郎</th>  <!-- 行見出し -->
    <td>95</td>
  </tr>
</table>
scope属性の値説明
col列全体の見出し
row行全体の見出し
colgroup列グループの見出し
rowgroup行グループの見出し

colspan / rowspan属性（セルの結合）

<!-- colspan：列方向に結合 / rowspan：行方向に結合 -->
<table>
  <tr>
    <th colspan="2">前期</th>   <!-- 2列分を結合 -->
    <th colspan="2">後期</th>
  </tr>
  <tr>
    <td>4月</td><td>5月</td>

  </tr>
  <tr>
    <td rowspan="2">継続</td>  <!-- 2行分を結合 -->
    <td>A</td>
    <td>B</td>
    <td>C</td>
  </tr>
  <tr>
    <td>D</td>
    <td>E</td>
    <td>F</td>
  </tr>
</table>

③ thead要素 / tbody要素 / tfoot要素

ポイント
・thead：テーブルのヘッダー行グループ（省略可・1つまで）
・tbody：テーブルのボディ行グループ（省略可・複数配置可・省略時は自動補完）
・tfoot：テーブルのフッター行グループ（省略可・1つまで・最後に配置） 
<table>
  <thead>
    <tr>
      <th>科目</th>
      <th>前期</th>
      <th>後期</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>HTML</td><td>90</td><td>95</td>
    </tr>
    <tr>
      <td>CSS</td><td>85</td><td>88</td>
    </tr>
    <tr>
      <td>JavaScript</td><td>78</td><td>82</td>
    </tr>
  </tbody>

  <!-- tfoot は tbody の後に配置（HTML Living Standard） -->
  <tfoot>
    <tr>
      <td>合計</td><td>253</td><td>265</td>
    </tr>
  </tfoot>
</table>

④ caption要素

ポイント
・caption 要素はテーブルのタイトル・説明を表します。
・table 要素の最初の子要素として配置します。
・1つの table に caption は1つだけ配置できます。
・終了タグは省略できます。 
<table>
  <!-- captionはtableの最初の子要素 -->
  <caption>2025年度 試験結果一覧</caption>
  <thead>
    <tr>
      <th>氏名</th>
      <th>スコア</th>
      <th>合否</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>山田太郎</td><td>92</td><td>合格</td>
    </tr>
  </tbody>
</table>

⑤ colgroup要素 / col要素

ポイント
・colgroup 要素は列のグループを定義します。
・col 要素は個々の列を表す空要素です。
・CSSでの列スタイル適用・span 属性による複数列のグループ化に使います。
・colgroup は caption の後・thead の前に配置します。 
<table>
  <caption>月別売上表</caption>

  <!-- colgroup：列グループの定義 -->
  <colgroup>
    <col style="background:#fef3c7;">  <!-- 1列目 -->
    <col span="2" style="background:#ebf8ff;">  <!-- 2〜3列目 -->
    <col style="background:#c6f6d5;">  <!-- 4列目 -->
  </colgroup>

  <thead>
    <tr>
      <th>月</th>
      <th>商品A</th>
      <th>商品B</th>
      <th>合計</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>1月</td><td>120</td><td>80</td><td>200</td>
    </tr>
    <tr>
      <td>2月</td><td>150</td><td>95</td><td>245</td>
    </tr>
  </tbody>
</table>

<!-- span属性：複数列をまとめて指定 -->
<colgroup>
  <col span="3">  <!-- 最初の3列をグループ化 -->
</colgroup>

⚠️ col要素は空要素
・col 要素は空要素のため終了タグはありません
・span 属性で複数列をまとめて指定できます（デフォルトは1）
・CSSで適用できるプロパティは限定的です（background / width / border / visibility など）

⑥ border属性の廃止（HTML Living Standard）

ポイント①：HTML Living Standardでは table要素の border属性は削除されている
⚠️ 試験重要ポイント①：border属性の廃止

HTML4では <table border="1"> のように border 属性でテーブルの枠線を表示していました。

しかし HTML Living Standard（現在の標準仕様）では table要素の border属性は削除されています。

❌ <table border="1">　← 現在の仕様では不正

✅ テーブルの枠線は CSS で指定します： 
<!-- ❌ 非推奨（HTML Living Standardでは削除） -->
<table border="1"> ... </table>

<!-- ✅ 推奨：CSSで枠線を指定 -->
<style>
  table { border-collapse: collapse; }
  th, td { border: 1px solid #ccc; padding: 8px; }
</style>
<table> ... </table>
✅ HTML属性 vs CSS の使い分け原則
・構造・意味 → HTML属性で指定
・見た目・スタイル → CSSで指定
・border / bgcolor / align / width（テーブル用） などの見た目に関する属性はすべてCSSに移行されています

⑦ 終了タグの省略まとめ

<!-- 終了タグを省略した書き方（文法的に正しい） -->
<table>
  <caption>省略デモ          <!-- </caption> 省略可 -->
  <colgroup>
    <col>                    <!-- col は空要素 -->
    <col>
                             <!-- </colgroup> 省略可 -->
  <thead>
    <tr>
      <th>列1            <!-- </th> 省略可 -->
      <th>列2            <!-- </th> 省略可 -->
                           <!-- </tr> 省略可 -->
                           <!-- </thead> 省略可 -->
  <tbody>
    <tr>
      <td>データ1        <!-- </td> 省略可 -->
      <td>データ2        <!-- </td> 省略可 -->
                           <!-- </tr> 省略可 -->
                           <!-- </tbody> 省略可 -->
  <tfoot>
    <tr>
      <td>フッター1      <!-- </td> 省略可 -->
      <td>フッター2      <!-- </td> 省略可 -->
                           <!-- </tr> 省略可 -->
                           <!-- </tfoot> 省略可 -->
</table>               <!-- </table> は省略不可 -->

table❌ 省略不可必須
caption✅ 省略可—
colgroup✅ 省略可—
col— 空要素終了タグなし
thead✅ 省略可—
tbody✅ 省略可省略時はブラウザが自動補完
tfoot✅ 省略可tbody・trの後・1つのみ
