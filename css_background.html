🖼️ CSS 背景 学習サンプル

background-color / background-image / background-clip / background-repeat / background-size / background-origin / background-position / background-attachment / background の使い方を学びましょう。

① background-color

ポイント
・要素の背景色を指定します。
・色の指定方法は16進数・キーワード・rgb()・hsl()などすべて使えます。
・transparent（透明）がデフォルト値です。 
.box { background-color: #9f7aea; }
.box { background-color: skyblue; }
.box { background-color: rgb(159, 122, 234); }
.box { background-color: rgba(159, 122, 234, 0.4); }
.box { background-color: hsl(270, 60%, 60%); }
.box { background-color: transparent; } /* デフォルト値 */

② background-image

ポイント①：背景画像は複数指定できる
・background-image にカンマ区切りで複数の画像を指定できます。
・最初に書いた画像が最前面に表示されます。
・url() で画像ファイルを、linear-gradient() などでグラデーションを指定できます。 
/* 単一の背景画像 */
.box {
    background-image: url("image.png");
}

/* グラデーション */
.box {
    background-image: linear-gradient(to right, #9f7aea, #4299e1);
}
.box {
    background-image: radial-gradient(circle, #9f7aea, #1a202c);
}

/* ポイント①：複数の背景画像（カンマ区切り） */
/* 最初に書いた画像が最前面に表示される */
.box {
    background-image:
        url("icon.png"),          /* 最前面 */
        url("pattern.png"),       /* 2番目 */
        linear-gradient(to bottom, #9f7aea, #4299e1); /* 最背面 */
}

/* none：背景画像なし */
.box { background-image: none; }

▼ 複数背景画像のデモ（グラデーションを重ねた例）

2つのグラデーションを重ねた背景
⚠️ 複数背景の重ね順
・カンマ区切りで複数指定した場合、先に書いたものが前面に来ます
・各背景に対して background-repeat / background-position / background-size もカンマ区切りで個別指定できます
・background-color は常に最背面に表示されます

③ background-repeat

ポイント
背景画像の繰り返し方法を指定します。デフォルトは repeat（縦横に繰り返し）です。 
.box { background-repeat: repeat; }    /* 縦横に繰り返し（デフォルト） */
.box { background-repeat: repeat-x; }  /* 横方向のみ繰り返し */
.box { background-repeat: repeat-y; }  /* 縦方向のみ繰り返し */
.box { background-repeat: no-repeat; } /* 繰り返しなし */
.box { background-repeat: space; }     /* 切れないように等間隔で配置 */
.box { background-repeat: round; }     /* 切れないようにサイズ調整して繰り返し */

④ background-position

ポイント
背景画像の表示位置を指定します。 キーワード・px・%・calc() などで指定できます。 
/* キーワード指定 */
.box { background-position: center; }          /* 中央 */
.box { background-position: top left; }        /* 左上 */
.box { background-position: bottom right; }     /* 右下 */
.box { background-position: center top; }      /* 上中央 */

/* 数値・パーセント指定（水平 垂直の順） */
.box { background-position: 50% 50%; }        /* center と同じ */
.box { background-position: 20px 40px; }       /* 左から20px・上から40px */
.box { background-position: 100% 0%; }         /* 右上 */

/* CSS3：4値指定（端からの距離を指定） */
.box { background-position: right 20px bottom 10px; }
/* 右端から20px・下端から10pxの位置 */

⑤ background-size

ポイント③：背景画像は表示サイズを指定できる
・background-size で背景画像の表示サイズを指定できます。
・cover と contain がよく使われます。 
/* キーワード指定 */
.box { background-size: auto; }      /* 元のサイズ（デフォルト） */
.box { background-size: cover; }     /* 要素全体を覆う（アスペクト比維持・はみ出しあり） */
.box { background-size: contain; }   /* 要素内に収める（アスペクト比維持・余白あり） */

/* 数値・パーセント指定（幅 高さの順） */
.box { background-size: 200px 100px; } /* 幅200px・高さ100px */
.box { background-size: 50%; }          /* 幅50%・高さはauto */
.box { background-size: 100% 100%; }   /* 要素いっぱいに引き伸ばす */

/* 複数背景の場合はカンマ区切りで指定 */
.box {
    background-image: url("a.png"), url("b.png");
    background-size: 50px 50px, cover;
}
auto（デフォルト）
cover
contain
30px 30px
100% 100%
50%
値説明
auto画像の元のサイズ（デフォルト）
coverアスペクト比を維持しつつ要素全体を覆う（はみ出しあり）
containアスペクト比を維持しつつ要素内に収める（余白あり）
幅 高さ具体的なサイズを指定（px / % / em など）

⑥ background-origin

ポイント②：背景の表示領域は変更できる
・background-origin は背景画像の描画開始位置（原点）を指定します。
・background-clip と組み合わせて使うことが多いです。 
.box { background-origin: padding-box; }  /* paddingの左上が原点（デフォルト） */
.box { background-origin: border-box; }   /* borderの左上が原点 */
.box { background-origin: content-box; }  /* contentの左上が原点 */
padding-box（デフォルト）
paddingの左上が原点
border-box
borderの左上が原点
content-box
contentの左上が原点

⑦ background-clip

ポイント②：背景の表示領域は変更できる
・background-clip は背景（色・画像）をどの領域まで表示するかを指定します。
・text 値（CSS4）を使うとテキストの形に背景を切り抜けます。 
.box { background-clip: border-box; }   /* border領域まで表示（デフォルト） */
.box { background-clip: padding-box; }  /* padding領域まで（borderは透明） */
.box { background-clip: content-box; }  /* content領域のみ */

/* text：テキストの形に背景を切り抜く（CSS4） */
.text-clip {
    background-image: linear-gradient(to right, #9f7aea, #4299e1);
    background-clip: text;
    -webkit-background-clip: text; /* Safari/Chrome用 */
    color: transparent;
}

⑧ background-attachment

ポイント
背景画像をスクロールに対して固定するかどうかを指定します。 
.box { background-attachment: scroll; }  /* 要素と一緒にスクロール（デフォルト） */
.box { background-attachment: fixed; }   /* ビューポートに対して固定（パララックス効果） */
.box { background-attachment: local; }   /* 要素内のスクロールに追従 */
値説明用途
scroll要素と一緒にスクロール（デフォルト）通常の背景
fixedビューポートに対して固定パララックス効果・ヒーロー画像
local要素内のスクロールに追従スクロール可能なコンテナ

⑨ background（一括指定）

ポイント
・background は背景関連プロパティをまとめて指定できるショートハンドです。
・複数背景の場合はカンマ区切りで指定し、background-color は最後のレイヤーのみ指定できます。
・background-size は background-position の後に / で区切って指定します。 
/* background ショートハンドの書式 */
/* background: color image repeat attachment position / size origin clip; */

/* 基本的な例 */
.box {
    background: #9f7aea;
}
.box {
    background: url("bg.png") no-repeat center / cover;
}
.box {
    background: linear-gradient(to right, #9f7aea, #4299e1);
}

/* position / size の指定（/ で区切る） */
.box {
    background: url("icon.png") no-repeat right 10px center / 40px 40px;
}

/* 複数背景（カンマ区切り）・background-colorは最後のレイヤーに */
.box {
    background:
        url("icon.png") no-repeat center top / 30px,
        url("pattern.png") repeat,
        #f0f4f8;  /* background-colorは最後のレイヤーのみ */
}
⚠️ background ショートハンドの注意点
・background-size は background-position の直後に / で区切って指定します
・ショートハンドで指定しなかったプロパティは初期値にリセットされます
・background-color は複数背景の最後のレイヤーにのみ指定できます
・background-origin と background-clip を1値で指定すると 両方に同じ値が設定されます

.hero {
