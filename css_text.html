① text-shadow

ポイント
・テキストに影を付けます。
・水平オフセット 垂直オフセット ぼかし半径 色 の順で指定します。
・カンマ区切りで複数の影を重ねることができます。 
/* text-shadow: 水平 垂直 ぼかし 色 */
h1 { text-shadow: 2px 2px 4px rgba(0,0,0,0.4); }

/* ぼかしなし（シャープな影） */
p  { text-shadow: 3px 3px 0 #9f7aea; }

/* 複数の影をカンマ区切りで指定 */
h2 {
    text-shadow:
        1px 1px 0 #9f7aea,
        2px 2px 0 #4299e1,
        3px 3px 0 #48bb78;
}

/* 光彩（グロー）効果 */
.glow { text-shadow: 0 0 10px #9f7aea, 0 0 20px #9f7aea; }

/* none：影なし */
p  { text-shadow: none; }

▼ text-shadow のデモ

2px 2px 4px rgba(0,0,0,0.4) テキストシャドウ
3px 3px 0 #9f7aea（シャープ） テキストシャドウ
複数の影（3色） テキストシャドウ
グロー効果 テキストシャドウ

② text-decoration 関連

ポイント①：下線・上線・取消線は、色や線種が指定できる
・text-decoration は下線・上線・取消線などの装飾を指定します。
・CSS3以降は 色（color）・線種（style）・太さ（thickness） を個別に指定できます。 
/* text-decoration ショートハンド */
a  { text-decoration: underline; }
p  { text-decoration: none; }
del{ text-decoration: line-through; }
p  { text-decoration: overline; }

/* 複数の線を同時指定 */
p  { text-decoration: underline overline; }

/* ポイント①：色・線種・太さを個別指定（CSS3以降） */
a {
    text-decoration-line:      underline;       /* 線の種類 */
    text-decoration-color:     #9f7aea;         /* 線の色 */
    text-decoration-style:     wavy;            /* 線の形状 */
    text-decoration-thickness: 2px;             /* 線の太さ */
}

/* ショートハンドで一括指定 */
a { text-decoration: underline #9f7aea wavy 2px; }

/* text-decoration-style の値 */
.solid  { text-decoration: underline solid; }   /* 実線（デフォルト） */
.double { text-decoration: underline double; }  /* 二重線 */
.dotted { text-decoration: underline dotted; }  /* 点線 */
.dashed { text-decoration: underline dashed; }  /* 破線 */
.wavy   { text-decoration: underline wavy; }    /* 波線 */

/* text-underline-offset：下線の位置を調整 */
a { text-underline-offset: 4px; }

▼ text-decoration のデモ

underline solid（デフォルト） テキスト装飾
underline #9f7aea wavy 2px テキスト装飾
underline dotted #e53e3e テキスト装飾
underline dashed #4299e1 3px テキスト装飾
line-through double #e53e3e テキスト装飾
overline #48bb78 solid 3px テキスト装飾
underline overline（複数） テキスト装飾
個別プロパティ説明主な値
text-decoration-line線の種類none / underline / overline / line-through
text-decoration-color線の色色の値（デフォルト：currentColor）
text-decoration-style線の形状solid / double / dotted / dashed / wavy
text-decoration-thickness線の太さauto / px / em など
text-underline-offset下線の位置調整auto / px / em など

③ word-break / hyphens

ポイント②：word-breakプロパティは行の折返しに関する設定をする
ポイント③：hyphensプロパティはハイフネーションに関する設定をする
・word-break：単語の途中で改行するかどうかを制御します。
・hyphens：行末でハイフン（-）を挿入して単語を分割するかどうかを制御します。 
/* word-break：行の折返し設定 */
p { word-break: normal; }    /* デフォルト：言語のルールに従う */
p { word-break: break-all; } /* 文字単位で折返し（CJK以外も） */
p { word-break: keep-all; }  /* CJKは単語単位で折返し（禁則処理強化） */
p { word-break: break-word; }/* 単語が収まらない場合のみ折返し */

/* hyphens：ハイフネーション設定 */
p { hyphens: none; }     /* ハイフネーションしない */
p { hyphens: manual; }   /* &shy; のある位置のみ（デフォルト） */
p { hyphens: auto; }     /* ブラウザが自動でハイフン挿入 */

/* &shy;（ソフトハイフン）：手動でハイフン位置を指定 */
/* HTMLに <p lang="en">super&shy;cali&shy;fragil...</p> と記述 */

▼ word-break のデモ（幅を制限した場合）

normal
superlongwordthatdoesnotfit
break-all
superlongwordthatdoesnotfit
keep-all（日本語）
この文章は禁則処理が強化されます
プロパティ値説明
word-breaknormal言語のデフォルトルールに従う（デフォルト）
break-all文字単位で折返し（英語の単語も途中で改行）
keep-allCJK文字を単語単位で折返し（禁則処理強化）
break-word単語が収まらない場合のみ途中で改行
hyphensnoneハイフネーションしない
manual&shy;（ソフトハイフン）の位置のみ（デフォルト）
autoブラウザが自動でハイフン挿入（lang属性が必要）
⚠️ hyphens: auto の注意点
・hyphens: auto を使うには HTML要素に lang 属性が必要です
・例：<html lang="en"> または <p lang="de">
・日本語（lang="ja"）では多くのブラウザで未対応です

④ white-space

ポイント
空白文字（スペース・タブ・改行）の扱い方と、テキストの折返しを制御します。 
p { white-space: normal; }    /* 連続空白を1つに・折返しあり（デフォルト） */
p { white-space: nowrap; }    /* 折返しなし（1行に表示） */
p { white-space: pre; }       /* 空白・改行をそのまま保持（折返しなし） */
p { white-space: pre-wrap; }  /* 空白・改行を保持しつつ折返しあり */
p { white-space: pre-line; }  /* 改行を保持・連続空白は1つに・折返しあり */
p { white-space: break-spaces; } /* pre-wrapに似るが行末空白も折返し対象 */

▼ white-space のデモ（入力テキスト：「Hello World\n改行あり」）

pre-wrap

値連続空白改行折返し
normal1つに圧縮1つの空白にあり
nowrap1つに圧縮1つの空白になし

⑤ text-align / vertical-align

/* text-align：水平方向の揃え */
p { text-align: left; }      /* 左揃え（デフォルト） */
p { text-align: right; }     /* 右揃え */
p { text-align: center; }    /* 中央揃え */

p { text-align: end; }       /* 書字方向の終了側（CSS3） */

/* vertical-align：インライン要素の垂直方向の揃え */
img  { vertical-align: baseline; }  /* ベースラインに揃え（デフォルト） */
span { vertical-align: top; }       /* 行の上端に揃え */
span { vertical-align: middle; }    /* 中央に揃え */

sup  { vertical-align: super; }     /* 上付き文字 */
sub  { vertical-align: sub; }       /* 下付き文字 */
span { vertical-align: 4px; }       /* 数値で指定 */

⑥ line-height / text-indent

/* line-height：行の高さ（行間） */
p { line-height: normal; }   /* ブラウザのデフォルト（約1.2） */
p { line-height: 1.8; }     /* 単位なし数値（推奨：フォントサイズの1.8倍） */
p { line-height: 24px; }    /* px指定 */
p { line-height: 150%; }    /* パーセント指定 */
p { line-height: 2em; }     /* em指定 */

/* text-indent：テキストの字下げ（最初の行のインデント） */
p { text-indent: 1em; }     /* 1文字分の字下げ */
p { text-indent: 2em; }     /* 2文字分の字下げ */
p { text-indent: -1em; }    /* 負の値（ぶら下げインデント） */
p { text-indent: 10%; }     /* 親要素の幅に対するパーセント */

▼ line-height のデモ

line-height: 1.0

行間が狭いテキストです。複数行になると読みにくくなります。

line-height: 1.8

行間が適切なテキストです。複数行になっても読みやすいです。

line-height: 3.0

行間が広すぎるテキストです。

▼ text-indent のデモ

text-indent: 2em を指定した段落です。最初の行だけ2文字分字下げされています。2行目以降は通常の位置から始まります。

text-indent: 0 の段落です。字下げなし。

✅ line-height の単位なし数値が推奨される理由
・line-height: 1.8（単位なし）は子要素に倍率として継承されます
・line-height: 1.8em（em付き）は計算済みの px 値として継承されるため、 子要素のフォントサイズが変わっても行間が変わらず意図しない結果になることがあります

⑦ letter-spacing / word-spacing

p { letter-spacing: normal; }  /* デフォルト */
p { letter-spacing: 0.1em; }  /* 文字間を広げる */

p { word-spacing: normal; }   /* デフォルト */
p { word-spacing: 0.5em; }   /* 単語間を広げる */
p { word-spacing: -0.1em; }  /* 単語間を詰める */

⑧ text-transform

ポイント
英字の大文字・小文字変換を指定します。HTMLのテキスト自体は変わりません。 
p { text-transform: none; }        /* 変換なし（デフォルト） */
p { text-transform: uppercase; }   /* すべて大文字 */
p { text-transform: lowercase; }   /* すべて小文字 */
p { text-transform: capitalize; }  /* 各単語の先頭を大文字 */
p { text-transform: full-width; }  /* 全角変換（CJK向け） */

▼ text-transform のデモ（元テキスト：「hello world css」）

none hello world css
uppercase hello world css
lowercase HELLO WORLD CSS
capitalize hello world css

⑨ direction / unicode-bidi

ポイント
・direction：テキストの書字方向（左から右 / 右から左）を指定します。
・unicode-bidi：Unicodeの双方向テキストアルゴリズムの制御を指定します。
・通常はHTMLの dir 属性で制御するのが推奨ですが、 CSSでも指定できます。 
/* direction：書字方向 */
p { direction: ltr; }  /* 左から右（Left To Right）デフォルト */
p { direction: rtl; }  /* 右から左（Right To Left）アラビア語・ヘブライ語など */

/* unicode-bidi：双方向テキストの制御 */
p { unicode-bidi: normal; }          /* デフォルト */
p { unicode-bidi: embed; }           /* 埋め込みレベルを開く */
p { unicode-bidi: bidi-override; }   /* direction の方向を強制 */
p { unicode-bidi: isolate; }         /* 周囲から独立した方向（CSS3） */
p { unicode-bidi: isolate-override; }/* isolate + bidi-override（CSS3） */
p { unicode-bidi: plaintext; }       /* Unicode P3アルゴリズム（CSS3） */

▼ direction のデモ

direction: ltr（デフォルト） Hello World / こんにちは
direction: rtl Hello World / こんにちは
⚠️ direction と unicode-bidi の注意点
・direction 単独では効果が限定的で、 unicode-bidi: bidi-override と組み合わせて使うことが多いです
・多言語対応では CSS より HTML の dir 属性や <bdi> / <bdo> 要素の使用が推奨されます
