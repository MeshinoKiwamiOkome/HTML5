🏷️ HTML5 グローバル属性 学習サンプル

すべてのHTML要素に共通して使えるグローバル属性の役割と使い方を学びましょう。

① 属性と要素のおさらい

重要事項：属性と要素の違い
グローバル属性を学ぶ前に、「要素」と「属性」の違いをしっかり確認しましょう。

🧱 要素（Element）

・開始タグ〜終了タグ全体のこと
・コンテンツに意味・構造を与える
・例：<p>、<h1>、<a>

<p>本文テキスト</p>
↑ これ全体が「p要素」

🔧 属性（Attribute）

・開始タグ内に記述する追加情報
・要素に性質・設定を付け加える
・例：class、id、href

<a href="url">リンク</a>
↑ href="url" が「属性」 
<!-- 要素・属性・属性値の関係 -->
<a href="https://example.com"
   class="btn-primary"
   id="top-link"
   target="_blank">
  リンクテキスト
</a>

<!-- ↑ a要素に4つの属性が指定されている -->
<!-- href / class / id / target はすべて「属性」 -->
<!-- "https://..." / "btn-primary" などは「属性値」 -->
⚠️ 属性の書き方ルール
・属性は開始タグ内にのみ記述できます（終了タグには書けません）
・属性値はダブルクォート（"）で囲むのが推奨です
・同じ属性を同じ要素に2回以上指定することはできません
・値を持たない論理属性（disabled、checked など）は 値を省略して属性名だけ書けます

② グローバル属性とは

ポイント
・グローバル属性とは、すべてのHTML要素に共通して指定できる属性のことです。
・要素固有の属性（href は a 要素専用など）とは異なります。
・代表的なグローバル属性を以下に示します。
class
種類・分類を示す名前
id
固有の識別名
lang
言語コード
title
補足情報・ツールチップ
dir
テキスト方向
tabindex
タブ順序
style
インラインCSS
hidden
要素を非表示にする
data-*
カスタムデータ属性
contenteditable
編集可能にする
draggable
ドラッグ可否
accesskey
キーボードショートカット

③ class属性

ポイント①：class属性の値には「種類や分類を示す名前」を指定する
・class 属性はCSSのスタイル適用やJavaScriptの操作対象を指定するために使います。
・複数の要素に同じclass名を指定できます（id属性との大きな違い）。
・1つの要素に複数のclass名をスペース区切りで指定できます。 
<!-- 基本的な使い方 -->
<p class="highlight">強調テキスト</p>
<p class="highlight">同じclassを複数要素に指定できる</p>

<!-- 複数のclass名をスペース区切りで指定 -->
<button class="btn btn-primary btn-large">
  送信する
</button>

<!-- CSSでの参照（.クラス名） -->
<!-- .highlight { background: yellow; } -->
<!-- .btn-primary { background: #4299e1; color: #fff; } -->

<!-- JavaScriptでの参照 -->
<!-- document.querySelectorAll('.highlight') -->
⚠️ class属性の命名ルール
・見た目ではなく役割・種類・分類を示す名前にする
❌ class="red-text"　← 見た目（色）を表す名前（避けるべき）
✅ class="error-message"　← 役割を表す名前（推奨）

・数字で始めることはできません（class="1st" は不正）
・スペースは区切り文字として使われるため、class名にスペースは含められません
✅ class属性の主な用途
・CSSでスタイルを適用する（最も一般的）
・JavaScriptで複数要素をまとめて操作する
・コンポーネントの種類・状態を表す（is-active、is-hidden など）

④ id属性

ポイント②：id属性の値には「ほかの要素と重複しない固有の名前」を指定する
・id 属性は文書内で一意（ユニーク）な識別子を指定します。
・同じid値を複数の要素に指定することはできません。
・ページ内リンク（アンカーリンク）のターゲットとして使えます。 
<!-- 基本的な使い方 -->
<h2 id="section-intro">はじめに</h2>

<!-- ページ内リンク（アンカーリンク）のターゲット -->
<a href="#section-intro">「はじめに」へ移動</a>

<!-- CSSでの参照（#id名） -->
<!-- #section-intro { color: #2b6cb0; } -->

<!-- JavaScriptでの参照 -->
<!-- document.getElementById('section-intro') -->

<!-- ❌ 同じid値を複数の要素に指定してはいけない -->
<p id="intro">段落1</p>
<p id="intro">段落2</p>  <!-- 不正：id重複 -->
比較項目class属性id属性
文書内での重複 ✅ 複数要素に同じ値を指定可 ❌ 文書内で一意（重複不可）
1要素への複数指定 ✅ スペース区切りで複数指定可 ❌ 1つの値のみ
CSSセレクタ .クラス名 #id名
ページ内リンク ❌ 使用不可 ✅ href="#id名" で参照可
主な用途 スタイル・グループ操作 一意識別・アンカー・JS操作

⑤ lang属性

ポイント
・lang 属性は要素内のコンテンツの言語を指定します。
・<html> 要素に指定すると文書全体の言語になります。
・スクリーンリーダーの読み上げ言語・検索エンジンの言語判定に影響します。
・値は BCP 47言語タグ（ISO 639-1言語コード）を使います。 
<!-- 文書全体の言語指定 -->
<html lang="ja">

<!-- 部分的に異なる言語を指定 -->
<p>日本語のテキストの中に
  <span lang="en">English text</span>
が含まれる場合。
</p>

<!-- 引用文の言語指定 -->
<blockquote lang="fr">
  Je pense, donc je suis.
</blockquote>
言語コード言語
ja日本語
en英語
zh-CN中国語（簡体字）
zh-TW中国語（繁体字）
ko韓国語
frフランス語
deドイツ語
arアラビア語

⑥ title属性

ポイント
・title 属性は要素に関する補足情報を指定します。
・多くのブラウザではマウスオーバー時にツールチップとして表示されます。
・<abbr> 要素と組み合わせて略語の正式名称を示すのが代表的な用途です。 
<!-- ツールチップとして表示 -->
<p title="クリックして詳細を表示">
  この段落にマウスを乗せてください。
</p>

<!-- abbr要素と組み合わせた略語の説明 -->
<p>
  <abbr title="HyperText Markup Language">HTML</abbr>
  は文書構造を定義するマークアップ言語です。
</p>

<!-- リンクの補足説明 -->
<a href="https://html5exam.jp/"
   title="HTML5プロフェッショナル認定試験の公式サイトへ">
  試験公式サイト
</a>

▼ 実際の表示（マウスを乗せるとツールチップが表示されます）

HTML は文書構造を定義するマークアップ言語です。

この文にマウスを乗せてみてください（スマホでは表示されません）

⚠️ title属性の注意点
・スマートフォンではツールチップが表示されません
・スクリーンリーダーによっては読み上げない場合があります
・重要な情報はtitle属性だけに頼らず、本文にも記載しましょう

⑦ dir属性

ポイント
・dir 属性はテキストの書字方向（左→右 / 右→左）を指定します。
・アラビア語・ヘブライ語など右から左に書く言語に対応するために使います。 
<!-- 左から右（デフォルト） -->
<p dir="ltr">Left to Right: Hello, World!</p>

<!-- 右から左（アラビア語・ヘブライ語など） -->
<p dir="rtl">Right to Left: مرحبا بالعالم</p>

<!-- 自動判定 -->
<p dir="auto">ブラウザが自動で方向を判定します</p>

▼ 実際の表示

ltr（左→右）: Hello, World! HTML5学習中

rtl（右→左）: Hello, World! HTML5学習中

値説明
ltrLeft to Right（左から右）。日本語・英語など
rtlRight to Left（右から左）。アラビア語・ヘブライ語など
autoコンテンツに基づいてブラウザが自動判定

⑧ tabindex属性

ポイント
・tabindex 属性はキーボードの Tab キーによるフォーカス順序を制御します。
・通常フォーカスできない要素（div、span など）にもフォーカス可能にできます。 
<!-- tabindex="0"：通常のタブ順序に追加 -->
<div tabindex="0">Tabキーでフォーカス可能なdiv</div>

<!-- tabindex="-1"：Tabキーでは到達しないがJSでフォーカス可能 -->
<div tabindex="-1" id="modal">モーダルダイアログ</div>

<!-- tabindex="1以上"：明示的な順序指定（非推奨） -->
<input tabindex="3" type="text">  <!-- 3番目にフォーカス -->
<input tabindex="1" type="text">  <!-- 1番目にフォーカス -->
<input tabindex="2" type="text">  <!-- 2番目にフォーカス -->

⑨ カスタムデータ（data-*）属性

ポイント③：HTML5以降では data-〇〇〇 の書式で独自の属性を追加できる
・data-* 属性はHTML要素に独自のデータを埋め込むための仕組みです。
・data- の後に任意の名前を付けて使います。
・JavaScriptから dataset プロパティで簡単にアクセスできます。
・HTMLの標準属性を汚染せずに独自データを持てます。 
<!-- data-*属性の基本 -->
<button data-user-id="12345"
        data-action="delete"
        data-confirm-msg="本当に削除しますか？">
  削除する
</button>

<!-- 商品カードへの適用例 -->
<div class="product-card"
     data-product-id="P-001"
     data-category="electronics"
     data-price="29800"
     data-in-stock="true">
  <h3>商品名</h3>
  <p>29,800円</p>
</div>

<!-- JavaScriptからのアクセス -->
<!-- const card = document.querySelector('.product-card'); -->
<!-- card.dataset.productId  → "P-001" -->
<!-- card.dataset.category   → "electronics" -->
<!-- card.dataset.price      → "29800" -->
<!-- card.dataset.inStock    → "true" -->
<!-- ※ ハイフンはキャメルケースに変換される -->
<!-- data-product-id → dataset.productId -->

▼ data-*属性のデモ（ボタンをクリックしてください）

← ボタンをクリックするとdata-*属性の値を表示します
⚠️ data-*属性の命名ルール
・必ず data- で始める
・data- の後に少なくとも1文字以上の名前が必要
・名前に大文字は使えません（data-userId は不正）
・ハイフン区切りを使います（data-user-id ✅）
・JavaScriptの dataset ではハイフンがキャメルケースに変換されます
　（data-user-id → dataset.userId）
