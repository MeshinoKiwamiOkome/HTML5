16進数・キーワード・rgb()・hsl()・opacity など、CSSの色指定の方法を学びましょう。

① 色を示す値：16進数

ポイント
16進数（HEX）は #RRGGBB の形式で赤・緑・青の各成分を 00〜FF（0〜255）で指定します。

16進数の構造

RR赤（Red）
GG緑（Green）
BB青（Blue）
→ 例：
#4299e1青色

BB青
AA透明度
→ 例：
#4299e180半透明青 
/* 6桁の16進数（#RRGGBB） */
color: #ff0000;   /* 赤 */
color: #00ff00;   /* 緑 */
color: #0000ff;   /* 青 */
color: #ffffff;   /* 白 */
color: #000000;   /* 黒 */
color: #4299e1;   /* 青系 */

/* 3桁の省略形（#RGB）：各桁を2つ並べたものと同じ */
color: #f00;      /* = #ff0000（赤） */
color: #0f0;      /* = #00ff00（緑） */
color: #fff;      /* = #ffffff（白） */
color: #000;      /* = #000000（黒） */

/* 8桁（#RRGGBBAA）：CSS4で追加・アルファ値付き */
color: #ff000080; /* 半透明の赤（80 = 128/255 ≈ 50%） */
color: #4299e1cc; /* 80%不透明の青 */

/* 4桁（#RGBA）：3桁の省略形にアルファ値を追加 */
color: #f008;     /* = #ff000088（半透明の赤） */

⚠️ 3桁省略形のルール
3桁に省略できるのは、各桁が同じ文字の繰り返しの場合のみです。
✅ #aabbcc → #abc（省略可）
❌ #a1b2c3 → 省略不可（各桁が異なる）

② 色を示す値：キーワード

ポイント①：CSS3からはtransparentがすべての箇所で使用可能
CSS3以降では、色の値が指定可能なすべての箇所で transparent（完全透明）が使用できます。 
/* 基本的なカラーキーワード */
color: red;
color: green;
color: blue;
color: white;
color: black;
color: gray;
color: orange;
color: purple;

/* 特殊なキーワード */
color:            transparent; /* 完全透明（CSS3以降すべての箇所で使用可） */
background-color: transparent; /* 背景を透明に */
border-color:     transparent; /* 枠線を透明に */
color:            currentColor; /* 現在のcolorプロパティの値を継承 */
color:            inherit;      /* 親要素の値を継承 */

green
blue

transparent
完全透明
✅ transparentの主な用途
・ボタンの背景を透明にする：background-color: transparent;
・グラデーションの透明端：linear-gradient(transparent, black)
・三角形の作成（borderで透明を使う）：border-color: transparent transparent red transparent;
・ホバー時の色変化アニメーション（transitionと組み合わせ）
⚠️ CSS2までのtransparent
CSS2では transparent は background-color にのみ使用可能でした。
CSS3以降では色の値が指定可能なすべての箇所（color・border-color・ outline-color・text-shadow など）で使用できます。

③ 色を示す値：rgb() / rgba()

ポイント②：不透明度は0.0〜1.0の単位をつけない数値で指定する
rgba() の第4引数（アルファ値）は 0.0（完全透明）〜 1.0（完全不透明）の単位なしの数値で指定します。 
/* rgb()：赤・緑・青を0〜255の整数で指定 */
color: rgb(255, 0, 0);       /* 赤 */
color: rgb(0, 128, 0);       /* 緑 */
color: rgb(66, 153, 225);    /* 青系 */
color: rgb(255, 255, 255);   /* 白 */
color: rgb(0, 0, 0);         /* 黒 */

/* パーセントでも指定可能 */
color: rgb(100%, 0%, 0%);    /* 赤 */

/* rgba()：アルファ値（不透明度）を追加 */
/* アルファ値は 0.0（完全透明）〜 1.0（完全不透明）の単位なし数値 */
color: rgba(255, 0, 0, 1.0);   /* 完全不透明の赤 */
color: rgba(255, 0, 0, 0.5);   /* 50%透明の赤 */
color: rgba(255, 0, 0, 0.0);   /* 完全透明 */
color: rgba(66, 153, 225, 0.8); /* 80%不透明の青 */

/* CSS4：スペース区切り・/ でアルファ値 */
color: rgb(255 0 0);           /* カンマなしも可（CSS4） */
color: rgb(255 0 0 / 0.5);     /* / でアルファ値（CSS4） */

⚠️ アルファ値の指定方法（試験頻出）
・アルファ値は 0.0〜1.0 の単位なしの数値
❌ rgba(255, 0, 0, 50%)　← CSS3では不正（CSS4では%も可）
✅ rgba(255, 0, 0, 0.5)　← 正しい（50%透明）

・0 = 完全透明、1 または 1.0 = 完全不透明
・0.5 の先頭の 0 は省略可：.5 でも可

④ 色を示す値：hsl() / hsla()

ポイント③：hslはhue（色相）・saturation（彩度）・lightness（明度）の意味
H
Hue（色相）
色の種類を角度で指定
0〜360（deg）
S
Saturation（彩度）
色の鮮やかさ
0%=グレー、100%=鮮やか
0%〜100%
L
Lightness（明度）
色の明るさ
0%=黒、50%=標準、100%=白
0%〜100%

色相（Hue）の色相環

0° 赤 60° 黄 120° 緑 180° シアン 240° 青 300° マゼンタ 360° 赤 
/* hsl()：色相・彩度・明度で指定 */
color: hsl(0,   100%, 50%);   /* 赤（色相0°） */
color: hsl(120, 100%, 50%);   /* 緑（色相120°） */
color: hsl(240, 100%, 50%);   /* 青（色相240°） */
color: hsl(0,   0%,   50%);   /* グレー（彩度0%） */
color: hsl(0,   100%, 0%);    /* 黒（明度0%） */
color: hsl(0,   100%, 100%);  /* 白（明度100%） */

/* 同じ色相で明度・彩度を変えてバリエーションを作る */
color: hsl(270, 100%, 30%);   /* 暗い紫 */
color: hsl(270, 100%, 50%);   /* 標準の紫 */
color: hsl(270, 100%, 70%);   /* 明るい紫 */
color: hsl(270, 50%,  50%);   /* くすんだ紫（彩度低め） */

/* hsla()：アルファ値を追加 */
color: hsla(270, 100%, 50%, 0.5); /* 50%透明の紫 */
color: hsla(0, 100%, 50%, 0.8);   /* 80%不透明の赤 */

/* CSS4：スペース区切り・/ でアルファ値 */
color: hsl(270 100% 50% / 0.5);   /* CSS4の書き方 */

✅ HSLのメリット
・色相（H）を変えるだけで別の色に変えられる
・明度（L）を調整するだけで同系色の明暗バリエーションが作れる
・デザインシステムで「同じ色相の薄い色・濃い色」を作るのに便利
・人間の感覚に近い色指定ができる（RGBより直感的）

⑤ colorプロパティ

/* colorプロパティ：テキストの色を指定 */
color: #2d3748;
color: red;
color: rgb(45, 55, 72);
color: hsl(220, 26%, 23%);

/* colorは子要素に継承される */
body { color: #2d3748; } /* 全体のテキスト色を設定 */

/* currentColor：現在のcolorプロパティの値を参照 */
.icon {
    color: #9f7aea;
    border: 2px solid currentColor; /* colorと同じ紫になる */
    fill: currentColor;              /* SVGのfillにも使える */
}

color: #9f7aea（紫）が指定されたテキスト

color: rgb(66, 153, 225)（青）が指定されたテキスト

color: hsl(120, 60%, 35%)（緑）が指定されたテキスト

color: rgba(255, 0, 0, 0.4)（40%不透明の赤）が指定されたテキスト

⑥ opacityプロパティ

ポイント②：不透明度は0.0〜1.0の単位をつけない数値で指定する
opacity プロパティは要素全体（子要素・テキスト・背景すべて）の 不透明度を指定します。 
/* opacityプロパティ：要素全体の不透明度 */
/* 0.0（完全透明）〜 1.0（完全不透明）の単位なし数値 */
opacity: 1;     /* 完全不透明（デフォルト） */
opacity: 0.8;   /* 80%不透明 */
opacity: 0.5;   /* 50%不透明（半透明） */
opacity: 0.2;   /* 20%不透明 */
opacity: 0;     /* 完全透明（見えないが領域は残る） */

⚠️ opacityの注意点
・opacity: 0 は要素が見えなくなりますが、 領域は残りクリックも可能です（display: none とは異なる）
・opacity は子要素に継承されるため、 親要素に設定すると子要素のテキストも透明になります
・背景だけ半透明にしたい場合は rgba() を使います
