🔤 CSS フォント 学習サンプル

Webフォント・font-family・font-size・font-weight・font-style・ font-variant・font ショートハンドを学びましょう。

① Webフォント（@font-face）

ポイント①：Webフォントを使用するには @font-face{} の書式を適用する
・@font-face でフォントファイルを定義し、 font-family プロパティで使用します。
・Google Fonts などの外部サービスを使う方法もあります。 
/* ===== @font-face の基本書式 ===== */
@font-face {
    font-family: "MyFont";            /* フォント名を定義 */
    src: url("fonts/myfont.woff2") format("woff2"),
         url("fonts/myfont.woff")  format("woff");
    font-weight: normal;
    font-style:  normal;
    font-display: swap;             /* フォント読み込み中の表示方法 */
}

/* 定義したフォントを使用 */
body {
    font-family: "MyFont", sans-serif;
}

/* ===== 複数ウェイトを定義する場合 ===== */
@font-face {
    font-family: "MyFont";
    src: url("fonts/myfont-bold.woff2") format("woff2");
    font-weight: bold;               /* 太字用フォントファイル */
}

/* ===== Google Fonts の使用例（link要素で読み込む） ===== */
/* HTML: <link href="https://fonts.googleapis.com/css2?
         family=Noto+Sans+JP&display=swap" rel="stylesheet"> */
body {
    font-family: "Noto Sans JP", sans-serif;
}
@font-face の主なプロパティ説明
font-familyフォントに付ける名前（必須）
srcフォントファイルのURL（必須）。複数指定でフォールバック
font-weightこのフォントファイルが対応するウェイト
font-styleこのフォントファイルが対応するスタイル
font-display読み込み中の表示方法（swap / block / fallback / optional）
unicode-rangeこのフォントを適用する文字の範囲
フォント形式format()の値対応状況
WOFF2"woff2"最新・最小サイズ。現在の主流
WOFF"woff"広くサポート。フォールバックに使用
TrueType"truetype"古いブラウザ向け
OpenType"opentype"古いブラウザ向け
⚠️ font-display: swap について
・swap：まずシステムフォントで表示し、Webフォント読み込み後に切り替える
・block：フォント読み込みまで非表示（最大3秒）
・fallback：非常に短い時間だけ非表示にし、その後フォールバックフォントを使用
・optional：ネットワーク状況が良い場合のみWebフォントを使用

② font-familyプロパティ

ポイント
・font-family はフォントの種類を指定します。
・カンマ区切りで複数指定でき、左から順に適用を試みます（フォールバック）。
・最後には必ず汎用ファミリー名を指定するのが推奨です。 
/* font-familyの基本 */
body {
    font-family: "Helvetica Neue", "Arial", sans-serif;
    /* ↑ 左から順に試みる。最後は汎用ファミリー名 */
}

/* スペースを含むフォント名はクォートで囲む */
h1 {
    font-family: "Times New Roman", serif;
}

/* スペースなしのフォント名はクォート不要（推奨はクォートあり） */
code {
    font-family: Consolas, Monaco, monospace;
}
汎用ファミリー名説明代表的なフォント
serif セリフ体（ひげ付き） Times New Roman / Georgia / 游明朝
sans-serif サンセリフ体（ひげなし） Arial / Helvetica / 游ゴシック
monospace 等幅フォント Consolas / Courier New / Monaco
cursive 筆記体・手書き風 Comic Sans MS / Brush Script MT
fantasy 装飾的なフォント Impact / Papyrus
system-ui OSのUIフォント San Francisco / Segoe UI

④ font-weightプロパティ

/* キーワード */
p { font-weight: normal; }   /* 400と同じ */
p { font-weight: bold; }     /* 700と同じ */
p { font-weight: lighter; } /* 親より1段階細く */
p { font-weight: bolder; }  /* 親より1段階太く */

/* 数値（100〜900、100刻み） */
p { font-weight: 100; }  /* Thin */
p { font-weight: 200; }  /* ExtraLight */
p { font-weight: 300; }  /* Light */
p { font-weight: 400; }  /* Regular（normalと同じ） */
p { font-weight: 500; }  /* Medium */
p { font-weight: 600; }  /* SemiBold */
p { font-weight: 700; }  /* Bold（boldと同じ） */
p { font-weight: 800; }  /* ExtraBold */
p { font-weight: 900; }  /* Black */

▼ font-weightの表示比較

font-weight: 100（Thin）

font-weight: 300（Light）

font-weight: 400（Regular / normal）

font-weight: 700（Bold / bold）

font-weight: 900（Black）

⚠️ font-weightの注意点
・フォントファイルが対応するウェイトのみ有効です
・対応するウェイトがない場合はブラウザが最も近いウェイトを選択します
・@font-face で複数ウェイトを定義することで細かい制御が可能です

⑤ font-styleプロパティ

p { font-style: normal; }   /* 通常（デフォルト） */
p { font-style: italic; }   /* イタリック体（専用フォントを使用） */
p { font-style: oblique; }  /* 斜体（通常フォントを傾ける） */
p { font-style: oblique 20deg; } /* 傾き角度を指定（CSS4） */

▼ font-styleの表示比較

font-style: normal（通常）

font-style: italic（イタリック体）

font-style: oblique（斜体）

✅ italic と oblique の違い
・italic：フォントに専用のイタリック体がある場合はそれを使用。 なければ oblique と同様に傾ける
・oblique：通常フォントを機械的に傾けたもの

⑥ font-variantプロパティ

ポイント②：font-variantは関連プロパティの値を一括指定するプロパティ
・font-variant は複数の font-variant-* サブプロパティの ショートハンドです。
・最も基本的な用途は small-caps（スモールキャップス）の指定です。 
/* font-variant の基本値 */
p { font-variant: normal; }      /* デフォルト */
p { font-variant: small-caps; }  /* スモールキャップス */
p { font-variant: none; }        /* すべてリセット */

/* font-variant は以下のサブプロパティの一括指定 */
p {
    font-variant-caps:        small-caps;  /* 大文字の扱い */
    font-variant-numeric:     oldstyle-nums; /* 数字のスタイル */
    font-variant-ligatures:   common-ligatures; /* 合字 */
    font-variant-alternates:  normal;
    font-variant-east-asian:  normal;
}

/* 一括指定の例 */
p { font-variant: small-caps oldstyle-nums; }

▼ font-variantの表示比較

font-variant: normal → Hello World CSS Typography 2025

font-variant: small-caps → Hello World CSS Typography 2025

サブプロパティ主な値説明
font-variant-caps small-caps / all-small-caps 大文字・スモールキャップスの制御
font-variant-numeric oldstyle-nums / lining-nums 数字のスタイル
font-variant-ligatures common-ligatures / no-common-ligatures 合字（fi, fl など）の制御
font-variant-east-asian jis78 / jis83 / ruby 東アジア文字の字体制御

⑦ fontプロパティ（ショートハンド）

ポイント③：fontプロパティは値の指定順序に注意する
・font ショートハンドは複数のフォント関連プロパティを一括指定します。
・font-size と font-family は必須で、順序も決まっています。
・省略した値はデフォルト値にリセットされます。

font ショートハンドの構文

font: font-style省略可 font-variant省略可 font-weight省略可 font-stretch省略可
       font-size必須 / line-height省略可（/の後に指定）
       font-family必須・最後に指定 ; 
/* font ショートハンドの使用例 */

/* 最小構成（必須のfont-sizeとfont-familyのみ） */
p { font: 16px sans-serif; }

/* font-size / line-height を指定（スラッシュで区切る） */
p { font: 16px / 1.8 sans-serif; }

/* font-weight と font-size / line-height と font-family */
p { font: bold 16px / 1.8 "Helvetica Neue", sans-serif; }

/* すべてを指定する場合の順序 */
p {
    font: italic          /* font-style */
          small-caps     /* font-variant */
          bold           /* font-weight */
          condensed      /* font-stretch */
          18px / 1.6     /* font-size / line-height */
          "Georgia", serif; /* font-family（最後・必須） */
}

/* ❌ 間違い例：font-sizeとfont-familyの順序が逆 */
p { font: sans-serif 16px; }  /* 無効 */

/* ❌ 間違い例：font-sizeを省略 */
p { font: bold sans-serif; }  /* 無効（font-sizeは必須） */

/* ⚠️ 省略した値はデフォルトにリセットされる */
p { font: 16px sans-serif; }
/* ↑ font-style / font-variant / font-weight は normal にリセット */

▼ font ショートハンドの表示比較

font: 14px serif

font: 16px / 2 sans-serif（行間2倍）

font: bold 18px / 1.4 sans-serif

font: italic bold 16px / 1.6 serif

font: small-caps bold 16px sans-serif

⚠️ font ショートハンドの重要ルール

① font-size と font-family は必須（省略不可）
② font-family は最後に指定する
③ line-height は font-size の後に スラッシュ（/）で区切って指定する
④ 省略したプロパティはデフォルト値にリセットされる
　（前に個別指定していた値も上書きされる）
⑤ font-style / font-variant / font-weight / font-stretch は font-size より前に指定する

font ショートハンドが一括指定するプロパティ

プロパティ必須位置省略時のデフォルト
font-style❌font-sizeより前normal
font-variant❌font-sizeより前normal
font-weight❌font-sizeより前normal
font-stretch❌font-sizeより前normal
font-size✅ 必須font-familyの直前—
line-height❌font-sizeの後 / で区切るnormal
font-family✅ 必須最後—
