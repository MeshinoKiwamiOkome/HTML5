📚 HTML5 セマンティック・インタラクティブ要素 学習サンプル

各要素の役割・使い方を実際の表示で確認しましょう。

① figure / figcaption

ポイント
・<figure> ― 画像・図・コードなど、本文から参照される独立したコンテンツを包む
・<figcaption> ― figure のキャプション（説明文）。figure の最初か最後の子要素として記述
・figure を削除しても本文の意味が成立するような補足的コンテンツに使う
▶ コードを見る 
<figure>
  <img src="sample.jpg" alt="サンプル画像">
  <figcaption>図1：サンプル画像の説明</figcaption>
</figure>

② details / summary

ポイント
・<details> ― クリックで開閉できる折りたたみコンテンツを作る
・<summary> ― details の見出し部分（クリック対象）。省略すると「詳細」と表示される
・open 属性を付けると初期状態で展開される
▶ HTML5 とは何ですか？（クリックして開く）

HTML5 は HyperText Markup Language のバージョン5です。
セマンティックな要素・マルチメディア対応・APIの充実が特徴です。

▶ open属性あり（最初から開いている）

このように open 属性を付けると、ページ読み込み時から展開された状態になります。

▶ コードを見る 
<details>
  <summary>クリックして開く</summary>
  <p>ここに詳細コンテンツ</p>
</details>

<!-- open属性で最初から展開 -->
<details open>
  <summary>最初から開いている</summary>
  <p>コンテンツ</p>
</details>

③ menu

ポイント
・<menu> ― コマンドやボタンのリストを表すセマンティックな要素
・<ul> と同じブロックレベルのリストだが、インタラクティブなコマンド群に使う意味的な違いがある
・ツールバーやアクションボタン群のマークアップに適している

ドキュメント操作メニュー：

    •
    ▶ コードを見る 
    <menu>
  <li><button>新規作成</button></li>
  <li><button>保存</button></li>
  <li><button>印刷</button></li>
</menu>

    ④ iframe

    ポイント
    ・<iframe> ― 別のHTMLページや外部コンテンツをページ内に埋め込む（インラインフレーム）
    ・src 属性で読み込むURLを指定
    ・title 属性はアクセシビリティのために必須
    ・sandbox 属性でセキュリティ制限をかけられる
    ・loading="lazy" で遅延読み込みが可能

    外部ページの埋め込み例：

    ※ セキュリティポリシーにより表示されない場合があります

    ▶ コードを見る 
    <iframe
  src="https://example.com"
  title="example.com の埋め込み"
  width="100%"
  height="200"
  loading="lazy"
  sandbox="allow-scripts allow-same-origin">
</iframe>

    ⑥ script / noscript

    ポイント
    ・<script> ― JavaScriptを埋め込む・外部JSファイルを読み込む
    ・defer 属性：HTML解析完了後にJS実行（DOMの準備後）
    ・async 属性：HTML解析と並行してJS取得・完了次第実行
    ・<noscript> ― JSが無効な環境向けの代替コンテンツを提供する

    ⏳ JavaScriptで書き換えられます...

    defer と async の違い：
    ・defer → HTML解析が終わってから実行（順序保証あり）
    ・async → ダウンロード完了次第すぐ実行（順序保証なし）
    ・どちらもなし → HTMLの解析をブロックして即実行

    ▶ コードを見る 
    <!-- インラインスクリプト -->
<script>
  console.log('Hello HTML5');
</script>

<!-- 外部ファイル読み込み -->
<script src="app.js" defer></script>
<script src="lib.js" async></script>

<!-- JS無効時の代替 -->
<noscript>
  <p>JavaScriptを有効にしてください。</p>
</noscript>

    ⑦ dialog

    ポイント
    ・<dialog> ― モーダルダイアログ・ポップアップを表すネイティブHTML要素
    ・open 属性があると表示される
    ・JS の .show() で通常表示、.showModal() でモーダル表示
    ・.close() で閉じる。::backdrop 疑似要素で背景をスタイリングできる

    📢 お知らせ

    これは <dialog> 要素を使ったネイティブモーダルです。
    JavaScriptの showModal() で呼び出しています。
    ▶ コードを見る 
    <button onclick="document.getElementById('myDialog').showModal()">
  ダイアログを開く
</button>

<dialog id="myDialog">
  <h3>お知らせ</h3>
  <p>ダイアログの内容</p>
  <button onclick="document.getElementById('myDialog').close()">
    閉じる
  </button>
</dialog>

    ⑧ template / slot

    ポイント
    ・<template> ― ページ読み込み時にはレンダリングされないHTMLの雛形。JSで複製して使う
    ・content プロパティで中身の DocumentFragment を取得できる
    ・<slot> ― Web Components（Shadow DOM）で使うコンテンツ挿入口
    ・name 属性で名前付きスロットを定義できる

    ⬇️ template要素からJSで動的に生成されたカード：

    slot の使用例（Web Components）： 

    <!-- Web Component定義内（Shadow DOM） -->
<template id="myComponent">
  <div>
    <slot name="title">デフォルトタイトル</slot>
    <slot name="body">デフォルト本文</slot>
  </div>
</template>

<!-- 使用側 -->
<my-component>
  <span slot="title">カスタムタイトル</span>
  <p slot="body">カスタム本文</p>
</my-component>
    ▶ template の基本コードを見る 
    <template id="myTemplate">
  <div class="card">
    <h3 class="title"></h3>
    <p class="body"></p>
  </div>
</template>

<script>
  const tmpl = document.getElementById('myTemplate');
  const clone = tmpl.content.cloneNode(true);
  clone.querySelector('.title').textContent = 'タイトル';
  document.body.appendChild(clone);
</script>

    ⑨ canvas

    ポイント
    ・<canvas> ― JSで2D/3Dグラフィックを描画するための領域
    ・width / height 属性で描画解像度を指定（CSSサイズとは別）
    ・getContext('2d') で2D描画コンテキストを取得
    ・canvas自体はピクセルベース（SVGはベクターベース）
    ・JS非対応環境向けにタグ内に代替テキストを書ける
    ※ お使いのブラウザはcanvasに対応していません

    ⬆️ JavaScriptで四角・円・三角・テキストを描画しています

    ▶ コードを見る 
    <canvas id="myCanvas" width="500" height="160">
  ※ canvasに対応していません（代替テキスト）
</canvas>

<script>
  const ctx = document.getElementById('myCanvas').getContext('2d');

  // 四角形を描く
  ctx.fillStyle = '#4299e1';
  ctx.fillRect(20, 40, 80, 80);

  // 円を描く
  ctx.beginPath();
  ctx.arc(200, 80, 45, 0, Math.PI * 2);
  ctx.fillStyle = '#f6ad55';
  ctx.fill();

  // テキストを描く
  ctx.fillStyle = '#2d3748';
  ctx.font = 'bold 14px sans-serif';
  ctx.fillText('Hello Canvas!', 300, 80);
</script>
